# Interactive React Portfolio

This project is a modern, interactive portfolio built with **React + Vite**.

## Features
- Hero section with quick contact information.
- Professional highlights cards.
- Skills badges.
- Experience timeline.
- Searchable projects section for better interaction.
- Education section.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

> Note: Update `src/resumeData.js` with exact resume values (email, phone, projects, and achievements) if you want 1:1 resume accuracy.
